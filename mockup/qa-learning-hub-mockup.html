Playwright / Locators

Locator Strategies

Playwright надає потужні методи для пошуку елементів на сторінці. Рекомендовано використовувати user-facing атрибути для стабільних тестів. 

# Рекомендовані локатори
from playwright.sync_api import Page

def test_login(page: Page):
    # По тексту
    page.get_by_text("Sign In").click()

    # По ролі
    page.get_by_role("button", name="Submit").click()

    # По placeholder
    page.get_by_placeholder("Email").fill("user@test.com")

    # По test-id (найстабільніший)
    page.get_by_test_id("login-button").click()
💡 Порада: Використовуйте get_by_test_id() для критичних елементів — це найстабільніший локатор, який не залежить від тексту чи стилів.
