Part 1

Hover Actions

Reveal hidden content

hover() - Mouse Over Elements

# Basic hover
page.locator("#menu-item").hover()

# Hover to reveal dropdown
page.locator(".dropdown-trigger").hover()
page.locator(".dropdown-menu").is_visible()  # Now visible

# Hover to reveal tooltip
page.locator("#info-icon").hover()
tooltip = page.locator(".tooltip")
assert tooltip.is_visible()

‚úÖ Always hover before interacting with elements that appear on hover!

Part 2

Click Variations

Beyond simple clicks

dblclick() - Double Click

# Double click to select text/word
page.locator("p").first.dblclick()

# Double click to open item
page.locator(".file-item").dblclick()

Right Click

# Right click for context menu
page.locator("#item").click(button="right")

# Click with modifiers
page.locator("a").click(modifiers=["Control"])  # Ctrl+Click
page.locator("a").click(modifiers=["Shift"])    # Shift+Click

Part 3

Drag and Drop

Move elements around

drag_to() - Simple Drag and Drop

# Drag source to target
source = page.locator("#draggable")
target = page.locator("#droppable")
source.drag_to(target)

# Drag file to upload area
file_element = page.locator(".file-item")
drop_zone = page.locator(".upload-zone")
file_element.drag_to(drop_zone)

Manual Drag and Drop

# More control over drag process
page.locator("#draggable").hover()
page.mouse.down()
page.locator("#droppable").hover()
page.mouse.up()

Part 4

Keyboard Actions

Shortcuts and typing

Keyboard Methods

# Press single key
page.keyboard.press("Enter")
page.keyboard.press("Escape")
page.keyboard.press("Tab")

# Type character by character
page.keyboard.press_sequentially("Hello World", delay=100)

# Keyboard shortcuts
page.keyboard.press("Control+A")  # Select all
page.keyboard.press("Control+C")  # Copy
page.keyboard.press("Control+V")  # Paste
page.keyboard.press("Control+Z")  # Undo

# Press key in specific element
page.locator("#input").press("Enter")

Common Keyboard Shortcuts

# Text editing
page.keyboard.press("Control+A")      # Select all
page.keyboard.press("Control+X")      # Cut
page.keyboard.press("Control+C")      # Copy
page.keyboard.press("Control+V")      # Paste
page.keyboard.press("Control+Z")      # Undo
page.keyboard.press("Control+Y")      # Redo

# Navigation
page.keyboard.press("Home")           # Start of line
page.keyboard.press("End")            # End of line
page.keyboard.press("PageUp")         # Page up
page.keyboard.press("PageDown")       # Page down

# Other
page.keyboard.press("F5")             # Refresh
page.keyboard.press("Control+F")      # Find

Part 5

Scroll Actions

Navigate the page

Scrolling Methods

# Scroll element into view
page.locator("#bottom-element").scroll_into_view_if_needed()

# Scroll to position
page.evaluate("window.scrollTo(0, 500)")

# Scroll by amount
page.evaluate("window.scrollBy(0, 500)")

# Scroll to top
page.evaluate("window.scrollTo(0, 0)")

# Mouse wheel scrolling
page.mouse.wheel(0, 500)  # Scroll down 500px
page.mouse.wheel(0, -500) # Scroll up 500px

Best Practices

‚úÖ DO

  • Hover before clicking hidden elements
  • Wait after hover for animations
  • Use drag_to() for simple cases
  • Test keyboard shortcuts
  • Scroll elements into view
  • Use appropriate wait strategies

‚ùå DON'T

  • Assume elements are visible
  • Skip waiting after actions
  • Hardcode scroll positions
  • Forget browser differences
  • Use complex manual drag logic
  • Skip keyboard accessibility tests
