String Methods: Case Manipulation

# Case conversion methods text = "Welcome to Python Automation" print(text.upper()) # "WELCOME TO PYTHON AUTOMATION" print(text.lower()) # "welcome to python automation" print(text.title()) # "Welcome To Python Automation" print(text.swapcase()) # "wELCOME TO pYTHON aUTOMATION" # Automation example: Case-insensitive comparison page_title = "Login Page" expected = "login page" if page_title.lower() == expected.lower(): print("‚úì Title matches!")
üí° Pro Tip: Always use .lower() for case-insensitive comparisons in tests!

String Methods: Whitespace Handling

# Removing whitespace messy_text = " extra spaces " print(f"Original: '{messy_text}'") print(f"Strip: '{messy_text.strip()}'") # 'extra spaces' print(f"LStrip: '{messy_text.lstrip()}'") # 'extra spaces ' print(f"RStrip: '{messy_text.rstrip()}'") # ' extra spaces' # Automation example: Clean user input email = input("Enter email: ") email = email.strip().lower() # Remove spaces + lowercase print(f"Cleaned email: {email}")
‚ö†Ô∏è Common Issue: Web scraping often gives extra whitespace. Always use .strip() to clean text!

String Methods: Replace & Split

Replace

# Replace text url = "https://example.com/old-page" new_url = url.replace("old-page", "new-page") print(new_url) # Multiple replacements text = "Python is great. Python rocks." text = text.replace("Python", "JavaScript")

Split & Join

# Split into list csv = "John,Doe,30,Engineer" fields = csv.split(",") print(fields) # ['John', 'Doe', '30', 'Engineer'] # Join list to string words = ["Python", "is", "awesome"] sentence = " ".join(words) print(sentence) # "Python is awesome"
üéØ Automation Use: Parse CSV data, split URLs, build dynamic selectors!

String Slicing - Accessing Parts

text = "Python Programming" # Individual characters (0-indexed) print(text[0]) # 'P' - first character print(text[-1]) # 'g' - last character # Slicing [start:end] print(text[0:6]) # 'Python' print(text[7:]) # 'Programming' print(text[:6]) # 'Python' print(text[-11:]) # 'Programming' # Step slicing [start:end:step] print(text[::2]) # 'Pto rgamn' - every 2nd char print(text[::-1]) # 'gnimmargorP nohtyP' - reverse!
# Automation example: Extract product ID from URL url = "https://store.com/product-12345" product_id = url.split("-")[-1] # '12345' print(f"Product ID: {product_id}")

text = "Welcome to login" # Check substring print("login" in text) # True # Find position print(text.find("login")) # 11 # Count print(text.count("o")) # 3 # Start / End url = "user@test.com" print(url.startswith("user")) # True print(url.endswith(".com")) # True

Control Flow - Making Decisions

Control flow lets your program make decisions and execute different code based on conditions.

Without Control Flow

# Always does the same thing page_title = "Dashboard" print("Clicking dashboard link") print("Loading data")

‚ùå Not flexible

With Control Flow

# Responds to conditions if "Dashboard" in page_title: print("On dashboard - loading data") else: print("Not on dashboard - navigating")

‚úì Dynamic & smart

üéØ In Automation: Control flow makes tests intelligent - they adapt to different page states!

if Statements - Basic Decisions

# Basic if statement age = 20 if age >= 18: print("You are an adult") print("You can vote") print("Program continues") # Always executes
‚ö†Ô∏è Indentation is CRUCIAL!
# Correct - indented block if age >= 18: print("Adult") # ‚úì Inside if block # Wrong - no indentation if age >= 18: print("Adult") # ‚ùå IndentationError!

if-else Statements

# If-else: Two paths temperature = 25 if temperature > 30: print("It's hot outside!") else: print("The weather is pleasant") # Automation example: Check login status is_logged_in = True if is_logged_in: print("Proceeding to dashboard") else: print("Redirecting to login page")
üí° Use Case: Handle two possible outcomes - success or failure, visible or hidden, etc.

if-elif-else: Multiple Conditions

score = 85 if score >= 90: print("Grade: A - Excellent!") elif score >= 80: print("Grade: B - Good job!") elif score >= 70: print("Grade: C - Satisfactory") elif score >= 60: print("Grade: D - Needs improvement") else: print("Grade: F - Failed")
# Automation: Handle different HTTP status codes status_code = 200 if status_code == 200: print("‚úì Success") elif status_code == 404: print("‚úó Page not found") elif status_code == 500: print("‚úó Server error") else: print(f"Status: {status_code}")

Comparison Operators

Operator Meaning Example Result
== Equal to 5 == 5 True
!= Not equal to 5 != 3 True
> Greater than 5 > 3 True
< Less than 5 < 3 False
>= Greater than or equal 5 >= 5 True
<= Less than or equal 5 <= 3 False
‚ö†Ô∏è Common Mistake: Use == for comparison, not = (which is assignment)!

Logical Operators: AND

and - Both conditions must be True

age = 25 has_license = True if age >= 18 and has_license: print("You can drive!") else: print("You cannot drive")

Truth Table for AND

Condition 1 Condition 2 Result
True True True
True False False
False True False
False False False
# Automation: Check element is visible AND enabled is_visible = True is_enabled = True if is_visible and is_enabled: print("Element is clickable")

Logical Operators: OR

or - At least one condition must be True

day = "Saturday" if day == "Saturday" or day == "Sunday": print("It's the weekend!") else: print("It's a weekday")

Truth Table for OR

Condition 1 Condition 2 Result
True True True
True False True
False True True
False False False
# Automation: Check if error message contains keywords error = "Invalid username or password" if "username" in error or "password" in error: print("Login credentials error")

Logical Operators: NOT

not - Inverts the boolean value

is_raining = False if not is_raining: print("You don't need an umbrella") else: print("Take an umbrella!")

NOT Truth Table

Value not Value
True False
False True
# Combining logical operators age = 25 is_student = False if age >= 18 and not is_student: print("Regular adult pricing") elif age >= 18 and is_student: print("Student discount available")

Nested if Statements

You can put if statements inside other if statements

account_balance = 500 withdrawal_amount = 200 is_account_active = True if is_account_active: print("Account is active") if account_balance >= withdrawal_amount: print(f"Withdrawal of ${withdrawal_amount} approved") remaining = account_balance - withdrawal_amount print(f"Remaining balance: ${remaining}") else: print("Insufficient funds") else: print("Account is not active. Cannot withdraw")
üí° Tip: Too many nested ifs can be hard to read. Use logical operators (and, or) when possible!

Practical Tips for Automation

String Tips

  • Always .strip() scraped text
  • Use .lower() for comparisons
  • Prefer f-strings for readability
  • Use in for substring checks
  • .split() is your friend for parsing

Control Flow Tips

  • Check most specific conditions first
  • Use and for required conditions
  • Use or for alternatives
  • Avoid deep nesting (max 2-3 levels)
  • Add comments for complex logic

Common Mistakes to Avoid

# ‚ùå Wrong: Assignment instead of comparison if status = 200: # SyntaxError! # ‚úì Correct: Use == for comparison if status == 200: # ‚ùå Wrong: Case-sensitive comparison if page_title == "dashboard": # Won't match "Dashboard" # ‚úì Correct: Case-insensitive if page_title.lower() == "dashboard":
