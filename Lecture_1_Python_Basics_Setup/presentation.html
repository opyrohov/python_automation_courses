Variables - Storing Data

Variables are containers that store data values.

Creating Variables

# No type declaration needed in Python! name = "Alice" age = 25 height = 5.6 is_student = True print(name) # Output: Alice print(age) # Output: 25 print(height) # Output: 5.6 print(is_student) # Output: True
üí° Pro Tip: Python automatically determines the data type based on the value you assign!

Data Types

Type Description Example
String (str) Text data "Hello", 'Python'
Integer (int) Whole numbers 42, -10, 0
Float Decimal numbers 3.14, -0.5, 2.0
Boolean (bool) True or False True, False

Checking Data Types

name = "Alice" print(type(name)) # Output: <class 'str'> age = 25 print(type(age)) # Output: <class 'int'>

Working with Strings

String Creation

# Single or double quotes first_name = "John" last_name = 'Doe' # Multi-line strings with triple quotes message = """Hello, Welcome to Python! Let's learn together."""

String Concatenation & Formatting

# Concatenation with + greeting = "Hello, " + first_name + "!" # F-strings (modern and recommended) message = f"My name is {first_name} {last_name}" print(message) # Output: My name is John Doe

String Indexing, Slicing & Methods

Indexing & Slicing

text = "Python" # Indexing (0-based) print(text[0]) # 'P' print(text[-1]) # 'n' (last) # Slicing [start:end] print(text[0:3]) # 'Pyt' print(text[2:]) # 'thon' print(text[:4]) # 'Pyth' # Length print(len(text)) # 6

Useful String Methods

name = " alice smith " # Case conversion print(name.upper()) # ' ALICE SMITH ' print(name.lower()) # ' alice smith ' print(name.title()) # ' Alice Smith ' # Whitespace removal print(name.strip()) # 'alice smith' # Replace & Split print(name.replace("alice", "bob")) print(name.split()) # ['alice', 'smith']
üí° Important: Strings are immutable - methods return new strings, they don't modify the original!

Operator Description Example Result
+ Addition 10 + 5 15

// Floor Division 17 // 5 3

Operator Precedence (Order of Operations)

Python follows PEMDAS rules (same as mathematics)

Priority Operators Description
1 (Highest) () Parentheses
2 ** Exponentiation
3 *, /, //, % Multiplication, Division
4 (Lowest) +, - Addition, Subtraction

Examples

# Without parentheses (multiplication first) result = 5 + 3 * 2 print(result) # Output: 11 (not 16!) # With parentheses (addition first) result = (5 + 3) * 2 print(result) # Output: 16 # Complex example result = 10 + 2 ** 3 * 2 print(result) # Output: 26 (2**3=8, 8*2=16, 10+16=26) # Always use parentheses for clarity! result = 10 + ((2 ** 3) * 2) print(result) # Output: 26 (much clearer!)
üí° Best Practice: When in doubt, use parentheses () to make your intentions clear!

Comparison Operators

Compare values and return True or False

a = 10 b = 20 print(a == b) # Equal to: False print(a != b) # Not equal to: True print(a > b) # Greater than: False print(a < b) # Less than: True print(a >= 10) # Greater than or equal: True print(a <= 5) # Less than or equal: False
üìù Note: Use == for comparison, not = (which is for assignment)

Logical Operators

Combine multiple conditions

AND - Both conditions must be True

age = 25 has_license = True can_drive = age >= 18 and has_license print(can_drive) # True

OR - At least one condition must be True

is_weekend = True is_holiday = False can_relax = is_weekend or is_holiday print(can_relax) # True

NOT - Inverts the boolean value

is_raining = False is_sunny = not is_raining print(is_sunny) # True

Boolean Context (Truthiness)

In Python, values can be evaluated as True or False in boolean contexts

‚ùå Falsy Values

# These evaluate to False: bool(0) # False bool(0.0) # False bool("") # False (empty string) bool([]) # False (empty list) bool({}) # False (empty dict) bool(None) # False bool(False) # False

‚úì Truthy Values

# These evaluate to True: bool(1) # True bool(-1) # True (any non-zero) bool("text") # True (any text) bool([1, 2]) # True (non-empty list) bool({"key": 1}) # True (non-empty dict) bool(True) # True

Practical Usage

# Check if a string has content name = "" if name: print("Name provided") else: print("Name is empty") # This will execute # Check if a list has items items = [] if items: print("Has items") else: print("No items") # This will execute
üí° Pro Tip: Use truthiness to write cleaner, more Pythonic code! Instead of if len(name) > 0:, simply use if name:

Getting User Input

The input() function lets you get data from users

Basic Input

# Get text input name = input("Enter your name: ") print(f"Hello, {name}!") # Output: # Enter your name: Alice # Hello, Alice!

‚ö†Ô∏è Important: input() Always Returns a String!

# Wrong - trying to do math with strings age = input("Enter your age: ") next_year = age + 1 # ‚ùå TypeError! # Correct - convert to integer first age = input("Enter your age: ") age = int(age) # Convert to number next_year = age + 1 print(f"Next year you'll be {next_year}") # Even better - convert on same line age = int(input("Enter your age: "))
üí° Remember: Always convert input() to the right type: int() for whole numbers, float() for decimals!

Variable Reassignment & Augmented Operators

Variables Can Change

# Initial assignment count = 10 print(count) # 10 # Reassign to new value count = 20 print(count) # 20 # Use current value to calculate new value count = count + 5 print(count) # 25

Augmented Assignment Operators (Shortcuts)

Long Form Short Form Description
x = x + 5 x += 5 Add and assign
x = x - 3 x -= 3 Subtract and assign
x = x * 2 x *= 2 Multiply and assign
x = x / 4 x /= 4 Divide and assign

Example

score = 100 score += 50 # score is now 150 score *= 2 # score is now 300 score -= 25 # score is now 275 print(score) # 275

Print Statements

Basic Printing

F-String Formatting (Recommended)

name = "Alice" age = 25 print(f"My name is {name} and I am {age} years old.") # With calculations inside print(f"Next year I'll be {age + 1} years old.")

Formatting Numbers

pi = 3.14159265 print(f"Pi: {pi:.2f}") # Output: Pi: 3.14 price = 1234.5 print(f"Price: ${price:,.2f}") # Output: Price: $1,234.50

Single-Line Comments

Variable Naming Conventions

‚úì Good Names

# Use lowercase with underscores user_name = "Alice" total_price = 99.99 is_active = True # Be descriptive student_count = 30 max_temperature = 100 # Constants in UPPERCASE MAX_RETRIES = 3 PI = 3.14159

‚úó Bad Names

# Don't start with numbers 1st_name = "Alice" # ‚ùå Error! # Don't use spaces user name = "Alice" # ‚ùå Error! # Avoid single letters x = "Alice" # ‚ùå Not descriptive tp = 99.99 # ‚ùå Unclear # Don't use Python keywords for = 5 # ‚ùå Reserved word!

Code Structure Best Practices

Proper Spacing

# Good: Space around operators total = price + tax average = (a + b + c) / 3 # Bad: No spaces (harder to read) total=price+tax average=(a+b+c)/3

Organize Your Code

# Section 1: Constants TAX_RATE = 0.08 SHIPPING_COST = 5.99 # Section 2: Variables product_name = "Laptop" base_price = 999.99 # Section 3: Calculations tax = base_price * TAX_RATE total = base_price + tax + SHIPPING_COST # Section 4: Output print(f"Total: ${total:.2f}")

Practical Example: Shopping Cart

# Shopping Cart Calculator # Product information item1_name = "Laptop" item1_price = 899.99 item1_quantity = 1 item2_name = "Mouse" item2_price = 25.50 item2_quantity = 2 # Calculate totals item1_total = item1_price * item1_quantity item2_total = item2_price * item2_quantity subtotal = item1_total + item2_total # Apply tax and calculate final total TAX_RATE = 0.08 tax_amount = subtotal * TAX_RATE final_total = subtotal + tax_amount # Display receipt print("=" * 40) print(f"{item1_name}: ${item1_price} x {item1_quantity} = ${item1_total:.2f}") print(f"{item2_name}: ${item2_price} x {item2_quantity} = ${item2_total:.2f}") print("-" * 40) print(f"Subtotal: ${subtotal:.2f}") print(f"Tax (8%): ${tax_amount:.2f}") print("=" * 40) print(f"TOTAL: ${final_total:.2f}")

Understanding Error Messages

Learning to read errors is a crucial skill! Python tells you exactly what went wrong.

Common Error Types

Error Type Meaning Example
SyntaxError Code structure is wrong print("Hello" (missing closing parenthesis)
NameError Variable doesn't exist print(x) when x was never defined
TypeError Wrong data type used "5" + 3 (can't add string and number)
ValueError Invalid value for type int("hello") (can't convert text to number)
IndentationError Incorrect spacing/tabs Inconsistent indentation in code blocks

Reading Error Messages

# Error example: Traceback (most recent call last): File "script.py", line 5, in <module> result = 10 + "5" TypeError: unsupported operand type(s) for +: 'int' and 'str' # What it tells you: # 1. File: script.py # 2. Line number: 5 # 3. Problem: Can't use + with int and str # 4. Solution: Convert string to int first!
üí° Pro Tip: Always read the error message from bottom to top! The last line tells you what went wrong, and the lines above show you where.

Common Beginner Mistakes

1. Confusing = and ==

# Assignment (=) x = 10 # Sets x to 10 # Comparison (==) x == 10 # Checks if x equals 10

2. Concatenating Different Types

# ‚ùå Error: Can't concatenate string and number age = 25 message = "Age: " + age # TypeError! # ‚úì Solution 1: Convert to string message = "Age: " + str(age) # ‚úì Solution 2: Use f-string message = f"Age: {age}"

3. Forgetting Quotes Around Strings

name = Alice # ‚ùå Error: Alice is not defined name = "Alice" # ‚úì Correct
