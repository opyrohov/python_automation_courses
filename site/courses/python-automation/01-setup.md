- Базові оператори

## Встановлення Python

1. Завантажте Python з [python.org](https://www.python.org/downloads/)
2. Встановіть з опцією "Add to PATH"
3. Перевірте встановлення:

```bash
python --version
```

## Hello World

```python
# Перша програма
print("Hello, World!")

# Змінні
name = "Python"
version = 3.12
is_awesome = True

print(f"Welcome to {name} {version}!")
```

## Типи даних

| Тип | Приклад | Опис |
|-----|---------|------|
| `str` | `"Hello"` | Рядок |
| `int` | `42` | Ціле число |
| `float` | `3.14` | Число з плаваючою точкою |
| `bool` | `True` | Булеве значення |
| `None` | `None` | Відсутність значення |
