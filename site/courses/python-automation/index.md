# Python Automation Course

Повний курс з автоматизації тестування — від основ Python до Playwright та Pytest.

## Огляд курсу

- **36 лекцій** з детальними поясненнями
- **100+ прикладів** коду
- **Практичні вправи** з розв'язками
- **Відео записи** кожної лекції
