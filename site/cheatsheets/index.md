## Як користуватися

1. **Bookmarks** — додайте в закладки потрібні сторінки
2. **Ctrl+F** — швидкий пошук по сторінці
3. **Copy** — копіюйте код прямо в проєкт

::: tip Порада
Використовуйте глобальний пошук `Ctrl+K` для швидкого знаходження потрібної інформації по всьому сайту.
:::
